# CistromeDB_Metadata


```
usage: extract_metadata_cistDB.py [-h] -n NUM [-c CONCAT] -p PATH -o OUTPUT

A script to get the the metadata information.

optional arguments:
  -h, --help            show this help message and exit
  -n NUM, --num NUM     The number to create a range to build the urls
  -c CONCAT, --concat CONCAT
                        The absolute path to the txt files generated in the
                        previous run to be concatenated and mapped into a
                        final df
  -p PATH, --path PATH  The absolute path to save the txt file if needed
  -o OUTPUT, --output OUTPUT
                        The absolute path to save the final df
```
